# Лабораторные работы по C++

Источник заданий: методические указания «Основы программирования на языке С++»


## Сборка всех сразу

Из каталога `cpp-labs/`:
```bash
make           # собрать все 7 лаб
./lab1/lab     # запустить любую
make clean     # удалить бинарники
```

## Сборка одной лабы

```bash
g++ -std=c++17 -O2 -Wall -Wextra lab1/main.cpp -o lab1/lab
./lab1/lab
```

## Что в каждой лабе

### Лаб. №1 — Линейные алгоритмы
Вычисление выражения `Y = sqrt(a² + b²) + sin(a) · cos(b)`.
Демонстрирует **оба** стиля ввода/вывода:
- `scanf` / `printf` (библиотека функций C);
- `cin` / `cout` с `iomanip` (библиотека классов C++).

### Лаб. №2 — Ветвление
Точка `(x, y)` относительно круга радиуса `R` с центром в начале координат:
внутри / на границе / вне области. Использует `if / else if / else`.

### Лаб. №3 — Итерационные циклы
Сумма ряда `S = Σ 1/n²` с заданной точностью `eps`, посчитанная одновременно
циклом `while` и циклом `do-while`. Сравнение с эталоном `π² / 6`.

### Лаб. №4 — Цикл `for` + одномерный массив
Из 15 целых чисел: количество отрицательных, количество нулей,
сумма положительных (вариант 1 из методички).

### Лаб. №5 — Двумерный массив + указатели
Сумма положительных элементов каждого столбца матрицы `A(m × n)`.
Доступ к элементам через арифметику указателей (вариант 1 из методички).

### Лаб. №6 — Строки
Подсчёт количества слов в строке и поиск
самого длинного слова. Используются `std::string` и `std::istringstream`.

### Лаб. №7 — Функции
Функция факториала и построенная над ней функция числа сочетаний `C(n, k)`.
Демонстрирует разделение программы на подпрограммы.
