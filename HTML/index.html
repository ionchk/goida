Практические работы по HTML

Курс «Веб-разработка», 17 заданий из методички «Практические работы по HTML». Каждый файл соответствует состоянию страницы после соответствующего задания.

  1. Задание №1. Создание простейшего файла HTML
  2. Задание №2. Управление расположением текста (тег BR)
  3. Задание №3. Специальные команды форматирования (P, NOBR)
  4. Задание №4. Выделение фрагментов текста (B, I, U)
  5. Задание №5. Размеры символов (H1..H6)
  6. Задание №6. Размер текущего шрифта (FONT SIZE)
  7. Задание №7. Гарнитура и цвет шрифта (FONT FACE, COLOR)
  8. Задание №8. Выравнивание текста по горизонтали (ALIGN)
  9. Задание №9. Цвет фона и текста (BGCOLOR, TEXT)
  10. Задание №10. Размещение графики (тег IMG)
  11. Задание №11. Атрибуты изображения (ALT, BORDER, …)
  12. Задание №12. Фоновое изображение (BACKGROUND)
  13. Задание №13. Создание таблицы (TABLE)
  14. Задание №14. Гипертекстовые связи (A)
  15. Задание №15. Ссылка на другой HTML-документ
  16. Задание №16. Ссылка-картинка на головную страницу
  17. Задание №17. Самостоятельное итоговое задание (мини-сайт «О группе»)

Источник заданий: методические указания «Практические работы по HTML».
