# Python

# По уч. литературе «Лабораторный практикум 1-6 по Python»

## Требования

# Минимум Python v3.10

## Запуск

```bash
python3 lab1/main.py
python3 lab2/main.py
python3 lab3/main.py 
python3 lab4/main.py
python3 lab5/main.py
python3 lab6/main.py
```

При первом запуске лаб №2 и №5 рядом со скриптом создаются вспомогательные
файлы (`books.csv`, `currency.xml`, `report.txt`) — они нужны для демонстрации
парсинга и регулярных выражений.

## Краткое описание

### Лаб. №1 — ANSI и вывод в консоль
Вариант 1: флаг Франции, узор «a», график `y = x²`, диаграмма процентного
соотношения чисел `< 0` и `> 0` в `sequence.txt`.

### Лаб. №2 — CSV и XML
Чтение `books.csv`, фильтрация книг до 150 рублей, поиск по автору, генерация
20 случайных библиографических ссылок в `bibliography.txt`. Парсинг
`currency.xml` через `xml.dom.minidom` и формирование словаря `Name → Value`.

### Лаб. №3 — Tkinter
Keygen-окно: формат ключа `XXXXX-XXXXX-XXXXX` (буквы A–Z и цифры 0–9).
Кнопки **Generate**, **Copy** (в буфер обмена), **Quit**, статусная строка.

### Лаб. №4 — Рюкзак
Сравнение жадного решения и точного 0/1-решения методом динамического
программирования с восстановлением списка предметов.

### Лаб. №5 — Регулярные выражения
Извлечение из текстового отчёта e-mail, телефонов в двух форматах, дат,
идентификаторов студенческих групп и почтовых индексов.

### Лаб. №6 — ООП
Иерархия классов `Person → Student / Teacher`, агрегирующий класс `Group`,
свойства, средние оценки, топ-N студентов. Демонстрирует инкапсуляцию,
наследование, полиморфизм и агрегацию.
